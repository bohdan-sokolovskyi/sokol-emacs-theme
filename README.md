# Sokol emacs theme

## Description
My custom theme for emacs

## Install
1) add to emacs init file
```lisp  
(add-to-list 'custom-theme-load-path "/path/to/folder/with/theme")
```
2) M-x customize-themes
3) choose `sokol-theme`
